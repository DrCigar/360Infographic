# 360 Infographic

POS360 Warehouse Operations — "Our Flow State." An interactive, on-brand infographic that follows a 360 Pro Register down the production line through three teams: Supply Chain, Pre-Deployments, and Deployments.

Built as a single self-contained HTML file (fonts via Google Fonts, logo embedded). Open index.html in any browser.

## View it live
Enable GitHub Pages (Settings -> Pages -> Deploy from branch -> main / root) and it will be served automatically, since the infographic is index.html.
